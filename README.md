# Todo Website

A simple and modern todo list website built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clear completed tasks
- Tasks are saved in your browser using local storage

## Files

- index.html: Main structure of the page
- styles.css: Styling for the interface
- script.js: Todo logic and local storage support

## How to Run

1. Open the project folder
2. Launch index.html in your browser

You can also run a local server if you prefer:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8000
```

## Notes

- The app works entirely in the browser
- Your tasks remain available even after refreshing the page
